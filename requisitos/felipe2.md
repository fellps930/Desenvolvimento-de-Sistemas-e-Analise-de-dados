## Requisitos Funcionais

1. O sistema deve ser capaz de verificar automaticamente incoerências relacionados a faltas nos cursos
2. O sistema deve ser capaz de verificar os feriados de Goiânia,Goiás e calcular automaticamente o planejamento de aula
3. O sistema deve conseguir verificar se os professores enviaram planos de aula

## Requisitos não Funcionais

1.O sistema deve ter uma interface intuitiva e limpa para o usuário conseguir utilizar ele de maneira produtiva
2.O sistema deve possuir uma maneira do usuário utilizar-se de suas credenciais para realizar login
3.O sistema deve ter grande parte de suas funções automatizada para evitar fadiga
